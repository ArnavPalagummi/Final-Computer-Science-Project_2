Project Objective

Project Name: Life Savers

Time Duration: 1 week: December 11th to December 19th, 2025

Class: Computer Science Essential - CSE

The objective of this project was meant to be a life saving aid for students and schools that unfortunately end up experiencing emergency situations. It includes drills such as Fire, Earthquake, Hurricane, Intruder, and Tornado, and what to do in these types of scenarios. For this project, me, and three other people worked together - Ismail, Abhijay, and Gianna were my partners. We all shared roles together and worked on the project equally. All of us coded on the same days since we all had computers of our own, as well as assigned parts of our own. I learned many things from this project as well. For example, unlike the previous project where I had only one partner, here I had three. This allowed me to improve my social skills and communication skills even further. We were all able to persuade each other as well as understand each other's opinions. Although we did have some challenges such as how we would incorporate an API into our project, we all discussed and made a group slideshow to plan everything out. We ended up overcoming all of our problems. I also improved my designing and coding skills since using an API required much more cognitive thinking then simple block - coding.
